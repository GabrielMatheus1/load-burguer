# Load Burguer

Projeto front-end estatico com uma animacao de loading inspirada em um hamburguer montado em camadas. A interface usa HTML e CSS para posicionar os elementos visuais e animar as partes do burger em loop.

## Visao geral

O projeto exibe:

- tres camadas visuais do hamburguer usando arquivos SVG;
- uma animacao continua com movimento suave;
- a identidade visual da marca posicionada na parte inferior da tela.

## Como executar

Como este projeto e estatico, nao ha etapa de build ou instalacao de dependencias.


## Como funciona

No arquivo style.css, a animacao desce-burguer move os elementos em diferentes tempos de atraso, criando o efeito de loading em camadas.
